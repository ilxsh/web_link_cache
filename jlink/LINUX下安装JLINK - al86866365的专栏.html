  • 确认是否安装libusb

      $ dpkg -l | grep libusb

     如果有libusb相关信息，说明已经安装了

<pre name="code" class="html">$ dpkg -l | grep libusb
ii  libusb-0.1-4                                   2:0.1.12-20                             userspace USB programming library
ii  libusb-1.0-0                                   2:1.0.9~rc3-2ubuntu1                    userspace USB programming library
ii  libusb-dev                                     2:0.1.12-20                             userspace USB









  • 安装readline






下载地址:








$ sudo tar -zxvf readline-7.0.tar.gz -C /usr/bin/
$ cd /usr/bin/
$ sudo ./configure 
$ sudo make
$ make install













  • 安装JLINK




下载地址：https://www.segger.com/downloads/jlink








<pre name="code" class="html">$ sudo tar -zxvf JLink_Linux_V610k_i386.tgz -C /usr/bin/
$ cd /usr/bin/
$ cd JLink_Linux_V610k_i386/
$ sudo cp libjlinkarm.so.6.* /usr/lib/
$ sudo cp 99-jlink.rules /etc/udev/rules.d/
$ sudo ldconfig
$ sudo ./JLinkExe 













第一次安装V610K版本，执行./JLinkExe始终报错




SEGGER J-Link Commander V6.10k
Could not open J-Link shared library. Exiting now.


没找到解决方法，改为安装V512j版本，步骤如上：



在执行ldconfig时，报错


          /sbin/ldconfig.real: /usr/lib/libjlinkarm.so.5 is not a symbolic link


参考http://blog.csdn.net/y249077293/article/details/8451888执行下列命令，解决问题


# rm libjlinkarm.so.5


# ln -s libjlinkarm.so.5.12.10 libjlinkarm.so.5


# ldconfig

















  • 测试是否安装成功














crazyuav@crazyuav:~/tools/JLink_Linux_V512j_i386$ ./JLinkExe 
  
DLL version V5.12j, compiled Jul 18 2016 18:52:40

Connecting to J-Link via USB...O.K.
Firmware: J-Link ARM V8 compiled Nov 28 2014 13:44:46
Hardware version: V8.00
S/N: 17935099
License(s): RDI,FlashDL,FlashBP,JFlash












  • 删除方式：


            






rm /usr/lib/ libjlinkarm.so.5.*    
rm /etc/udev/rules.d/99-jlink.rules 











  • 最简单的安装方式


       官网下载：.deb(相当于windows的exe),运行安装后


       在/etc/udev/rules.d/下有xx-jlink.rules    在/opt/下有SEGGER文件，segger文件如下


    


lrwxrwxrwx 1 root root   23 May 27 22:27 JLink -> /opt/SEGGER/JLink_V512g
drwxr-xr-x 5 root root 4096 Nov  6 04:07 JLink_V512g
猜测：若使用不同版本的JLink 只需软连接不同版本的JLink



          ln -s JLinkXXXXX JLink即可
