云服务器知识docker 看容器log命令

2019-09-05 11:35:10 分类：云服务器知识 编辑： editor 评论(0)（去评论）

docker 看容器log命令有什么用？docker logs [ID或者名字] 可以查看容器内部的标准输出。 -f: 让 docker logs 像使用 tail -f 一样来输出容器内部的标准输出。

通过docker logs命令可以查看容器的日志。

命令格式：

$ docker logs [OPTIONS] CONTAINER
Options:
--details 显示更多的信息
-f, --follow 跟踪实时日志
--since string 显示自某个timestamp之后的日志，或相对时间，如42m（即42分钟）
--tail string 从日志末尾显示多少行日志， 默认是all
-t, --timestamps 显示时间戳
--until string 显示自某个timestamp之前的日志，或相对时间，如42m（即42分钟）

举例来说

查看指定时间后的日志，只显示最后100行：

$ docker logs -f -t --since="2019-08-08" --tail=100 CONTAINER_ID

查看最近30分钟的日志:

$ docker logs --since 30m CONTAINER_ID

使用 tail 查看 log 尾部，效果和Linux的 tail-n20filename 一样，显示最后20行的内容。

$ docker logs --tail 20 <CONTAINER>
