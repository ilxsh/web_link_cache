pysvn安装流程

Posted on 2014-02-19 15:13  Andy2011  阅读(4407)  评论(0)  编辑  收藏

linux版：

新进入pysvn的安装目录

1 cd Source
2 python setup.py configure
3 make
4 cd ../Tests
5 make
6 cd Source
7 mkdir [YOUR PYTHON LIBDIR]/site-packages/pysvn
8 cp pysvn/__init__.py [YOUR PYTHON LIBDIR]/site-packages/pysvn
9 cp pysvn/_pysvn*.so [YOUR PYTHON LIBDIR]/site-packages/pysvn

windows版：

http://pysvn.tigris.org/servlets/ProjectDocumentList?folderID=1768

去这个地方下载win版的svn，特别注意：

1、当前系统使用的python版本，比如2.7

2、对应的svn客户端版本比如1.8.5；不确定到底是客户端版本还是服务器端版本号，但是我用客户端版本号是可行的

3、下载跟上述2者相对应的pysvn库版本

正常安装即可
