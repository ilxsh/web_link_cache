AUTOFS自动挂载NFS共享方法

  1. 系统环境ubuntu，安装：sudo apt-get install autofs
  2. 主要有两个配置文件
  3. 修改auto.master。/test是要挂载到的本地目录，/etc/auto.misc是挂载的配置。（名称可以修改）
  4. 修改/etc/auto.misc，autofstest和autofsnfs是在/test目录下自动创建一个挂载的目录，后面是使用nfs共享出来的目录。
  5. 验证。

    附录：nfs的安装和使用
  6. 安装nfs软件：sudo apt-get install nfs-kernel-server。
  7. 配置nfs，sudo vi /etc/exports。/autofstest是共享的目录，后面为共享给某个主机和各种参数。
  8. 使用showmount –e进行查看，如果出现如下报错，需要添加/etc/hosts，ip地址和主机名。
  9. 添加/etc/hosts解析
  10. 验证。
