Instantiating a Netlist Core in a Custom Peripheral
You might need to instantiate a netlist core in your custom peripheral (such as user_logic.vhd).
To Instantiate a Netlist Core in a Custom Peripheral
  1.  In the Microprocessor Peripheral Definition (MPD) file, set the OPTION STYLE parameter to MIX. The following is an example code snippet from an MPD file:
    	OPTION IPTYPE = PERIPHERAL 
	OPTION IMP_NETLIST = TRUE 
	OPTION HDL = VHDL 
	OPTION IP_GROUP = MICROBLAZE:PPC:USER 
	OPTION CORE_STATE = DEVELOPMENT 
	OPTION STYLE = MIX
  2.  Create a netlist folder inside your pcore directory.
    The pcore directory now contains four folders: data, devl, hdl, and netlist.
  3.  Include a Black Box Definition (BBD) file in the data folder. The correct syntax for the BBD file is as follows:
    	FILES
	blackbox1.ngc, blackbox2.ngc, blackbox3.edn
    The following syntax is incorrect:
    	FILES blackbox1.ngc, blackbox2.ngc, blackbox3.edn
  4.  Include all netlists in the netlist folder. Netlists should be placed in the netlist folder, rather than the hdl folder.
