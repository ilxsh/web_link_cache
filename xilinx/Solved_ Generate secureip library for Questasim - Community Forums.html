Re: Generate secureip library for Questasim

Jump to solution

Hi @stefaniecg,

 

Please go through following steps for simulation through Vivado:

1. Generate output products of IP.

2. Compile the simulation libraries by selecting simulator Questa

3. In simulation settings select Questa.

 

For simulating standalone:

1. Generate output products of IP.

2. Use "export_simulation" Tcl command. It will generate a folder export_sim.

3. Navigate to export_sim->Questa and use the scripts generated to simulate standalone.

 

Please use supported simulator version. Which vivado and questasim version you are using?

 

Thanks,
Arpan

Re: Generate secureip library for Questasim

Jump to solution

Hi @stefaniecg,

 

Refer to chapter 8 of this document on compiling libraries for third party simulators:
http://www.xilinx.com/support/documentation/sw_manuals/xilinx2015_4/ug900-vivado-logic-simulation.pdf

 

Or you can use this tcl command:
compile_simlib -force -library secureip -simulator questa -simulator_exec_path <path_to_questasim_executable>

Thanks,
Nupur
