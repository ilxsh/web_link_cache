dockerfile构建镜像，在执行CMD时，如何同时启动两个服务？

如下,希望的是这个容器同时运行cups打印服务和smb共享，已实现共享打印机服务 CMD ["/usr/sbin/smbd","-FS"] && ["cupsd"]
关注者

1 个回答

知乎用户

CMD [ "sh", "-c", "/usr/sbin/smbd -FS && /usr/sbin/cupsd -f"]

大致的原则, 串联的几个命令, 只要保持最后一个是在前台执行就可以了, 其中任何一个命令失败, 这个容器就启动失败了.
