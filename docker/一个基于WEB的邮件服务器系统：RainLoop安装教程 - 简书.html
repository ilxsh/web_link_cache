一个基于WEB的邮件服务器系统：RainLoop安装教程

三里桃花不及卿
0.3 2018.10.19 17:15 字数 678

简介

RainLoop是一个基于WEB的邮件系统，使用PHP开拓，通过RainLoop可以将多个email凑合在一齐，程序拥有简介的界面和全面的功能，支持SMTP+IMAP，是个人邮件客户真个最好程序。这里说下部署方式。

部署

1、搭建web环境

咱们可以用lnmp、lamp一键包或者宝塔之类的面板来搭建web环境。

lnmp部署方式可参考：https://www.moerats.com/archives/61/

宝塔部署方式可看：https://www.bt.cn/Download/btsoftlinux.html

2、上传RainLoop程序

搭建好web环境后，这里以lnmp环境为例，增加站点并解析，再上传RainLoop程序到站点目录，RainLoop官网下载地址：http://www.rainloop.net/downloads/

进入您的站点根目录，执行上面的command：

cd /home/wwwroot/www.yourdomain.com

wget http://www.rainloop.net/repository/webmail/rainloop-latest.zip

unzip rainloop-latest.zip

如果unzip解压文件时出现问题，这说明没部署unzip，因而解压前需要先执行command：apt-get install unzip或者yum install unzip便可。当然这里也可以用WinSCP等ftp工具完成程序上传步骤！

部署完成后访问http://www.yourdomain.com/?admin输入用户名：admin，密钥：12345进行登录。

设置

1、进入后台后率先会要求你改密钥，然后设置邮局言语、正题、缩略图、背景挑选、附件巨细等等。

2、在后台点击域名，你会看到RainLoop原始启用了Gmail，如果您需要使用outlook、qqemail等，可以一并启用了。或者增加别的email也可以。

3、伏笔VPS探索了，就不多说了。

说明

RainLoop用PHP开拓，一般情况下在PHP环境中便可以执行，偶然候并不需要数据库，应用程序不存储电子邮件，应用直接访问邮件服务器和RainLoop显现邮件便了，简单实用！性能方面完整取决于你的服务器内存和网速了，如果您需要启用邮件联络人，则需要用数据库（MySQL），详细可以自行折腾。当然咱们伏笔VPS使用的话，完整没有问题。
