Docker镜像无法被删除的解决方法

MaAiQiang 关注 0人评论 1945人阅读 2018-08-09 17:02:24

1.查看当前所有镜像

docker images

2.删除镜像ubuntu，提示报错，如下图（这里提示镜像有被容器再使用中）

Error response from daemon: conflict: unable to remove repository reference "ubuntu" (must force) - container 23107e15cef9 is using its referenced image 735f80812f90

解决方法如下：

1.先删除ubuntu的容器，会存在多个容器占用同一个镜像的问题，请重复删除即可（由以上报错信息可知：container ID:23107e15cef9 name:ubuntu)

docker rm 23107e15cef9

2.再删除ubuntu镜像，并查看所有镜像，已经成功删除该镜像

docker rmi ubuntu
