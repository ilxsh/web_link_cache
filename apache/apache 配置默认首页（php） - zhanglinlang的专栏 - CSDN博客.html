apache 配置默认首页（php）

2018年03月02日 11:15:59 没事偷着乐琅 阅读数：3936

在 apache 目录的 apache.conf 目录中：

<IfModule dir_module>
    DirectoryIndex index.html index.php hello.php
</IfModule>

apache 会按顺序搜索，
index.html
index.php
hello.php
如果找到有就直接返回。忽略后面的文件。
