Virtualbox虚拟机开机自启for Centos7

时间：2018-09-21 01:06:30      阅读：1296      评论：0      收藏：0      [点我收藏+]

标签：enabled   配置   roo   img   class   属性   开机自启   pat   ice   

本文源链接地址：https:www.93bok.com

1、设置自启动的两个环境变量
vim /etc/default/virtualbox

VBOXAUTOSTART_DB=/etc/vbox
VBOXAUTOSTART_CONFIG=/etc/vbox/vboxauto.conf
2、配置启动用户
vim /etc/vbox/vboxauto.conf

default_policy = deny
root= {
allow = true
}
3、修改vbox目录权限
chgrp vboxusers /etc/vbox/
chmod 1775 /etc/vbox/
4、设置dbpath属性
VBoxManage setproperty autostartdbpath /etc/vbox

VBoxManage list systemproperties

5、指定虚拟机自动启动
VBoxManage modifyvm lianni06 --autostart-enabled on
VBoxManage modifyvm centos6 --autostart-enabled on

（可以看到生成了以用户名开始的文件）

6、重启vboxauto服务
systemctl restart vboxautostart-service
7、重启宿主机查看虚拟机是否已经自动运行
