听说过MoinMoin Wiki吗？


传送门：MoinMoinWiki


看起来不怎么样？
实际上它大名鼎鼎，使用它作为wiki的各种官方网站包括：

  • Python
  • Ubuntu
  • Apahce
  • Debian
  • FreeBSD
  • ....

不喜欢画面风格？希望它支持markdown？希望这个wiki可以扩展开发插件？
来来来，它有这些market：MoinMoinExtensions


我自己就用它搭建的个人wiki，响应速度很快，而且它是基于python的，所以也很易于维护和修改。
