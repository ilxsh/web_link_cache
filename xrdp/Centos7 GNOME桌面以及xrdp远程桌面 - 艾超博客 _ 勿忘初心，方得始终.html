Centos7 GNOME桌面以及xrdp远程桌面

  • 2018-01-03
  • • 阅读: 1687
  • • Linux

前言

在上一篇文章中CentOS7 GNOME桌面以及VNC远程连接用到VNC远程，这里介绍下使用Xrdp远程桌面。Xrdp跟VNC性质是一样的，首先需要有图形桌面(即GNOME或者KDE，没有安装的在上一篇中有介绍)，但是Xrdp最大的有点是，远程的时候不需要安装额外的软件就可以远程连接，因为它是基于rdp的一种连接方式，大大方便了Windows平台用户。

安装Xrdp

  • 安装图形桌面(这里就不在说了，看上一篇文章吧)

  • 下载安装 epel REPO

    yum install -y epel-release

  • 安装Xrdp包

    yum install -y xrdp

  • 设置开机自动启动

    systemctl enable xrdp

  • 开启并且运行Xrdp服务

    systemctl start xrdp

  • 开放防火墙

    #开放3389端口
firewall-cmd --zone=public  --add-port=3389/tcp --permanent
#重启防火墙
systemctl restart firewalld.service
