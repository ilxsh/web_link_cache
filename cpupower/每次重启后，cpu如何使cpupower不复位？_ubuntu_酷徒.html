每次重启后，cpu如何使cpupower不复位？

    问题:

    我使用这个命令

    
sudo cpupower frequency-set -u 1.5G

    要使最大CPU频率达到 1.5 GHz，但重新启动后，将它的设置为默认。

    我如何在引导时设置这个？

    回答 1:

    任何容易和可逆的方法

    打开 gedit，然后新建一个空白文档，然后将它的粘贴到其中

    
sudo cpupower frequency-set -u 1.5G

    现在将它保存为 cpu.sh，. sh 非常重要并运行

    
sudo cp ~/cpu.sh/etc/init.d/

    使它可以执行

    
sudo chmod +x/etc/init.d/cpu.sh

    然后运行

    
update-rc.d cpu.sh defaults

    现在这个脚本应该在启动时作为 root 运行

    回答 2:

    查看/etc/default/cpupower,。尝试添加 max_freq="1.50Hz"

    要详细说明步骤，请运行 sudo nano/etc/default/cpupower，添加上面的行，然后按ctrl键保存，然后按ctrl键退出。

    /etc/default/有全局配置文件，在引导时读取。 希望能解决这个问题。
