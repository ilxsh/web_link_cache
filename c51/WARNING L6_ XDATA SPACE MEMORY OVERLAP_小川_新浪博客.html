WARNING L6: XDATA SPACE MEMORY OVERLAP

(2013-09-23 18:10:37)
转载▼
标签：

it

 分类： 单片机

    最近，调试代码的时候，突然间发现了一个奇怪的错误：

*** WARNING L6: XDATA SPACE MEMORY OVERLAP
    FROM:    0000H
    TO:      007FH

    后来才发现了错误的原因：extern unsigned int xdata can_rx[64] _at_ 0x0080;在声明重定义的时候，这个地方的地址不能指定地址，否则的话就会报错，我想原因可能和重定义的时候extern unsigned int a=0;不能这样在赋值一个道理吧。
