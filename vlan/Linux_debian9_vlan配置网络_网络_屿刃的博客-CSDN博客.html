实验前步

网关LNXRTR

实验配置

网关LNXRTR配置
第一步：apt install vlan												安装vlan
第二步：vim /etc/modules												打开modules配置文件
第三步：8021q														写入配置并保存退出
第四步：vim /etc/network/interfaces									打开网卡配置文件
第五步：
auto vlan10 vlan20 vlan30 ens33
iface ens33 inet static
address 104.54.65.60/27

iface vlan10 inet static
address 192.168.10.254/24
vlan-raw-device ens36
																	写入配置信息并保存退出
iface vlan20 inet static
address 172.16.10.254/24
vlan-raw-device ens36

iface vlan20 inet static
address 10.10.10.254/24
vlan-raw-device ens36
第六步：systemctl restart networking									重启网卡服务
第七步：ifdown ens33 && ifup ens33									重启网卡接口
