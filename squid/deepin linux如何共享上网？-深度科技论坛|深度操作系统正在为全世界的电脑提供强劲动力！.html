deepinuser17 发表于 2019-4-13 12:29:35 | 只看该作者
可以用Squid代理软件。

在Linux机器上安装squid软件。 设定允许其它机器访问。 在Windows机器上设置浏览器代理。

1. 安装squid

  1. $ sudo apt install squid
复制代码


2. 设置squid, 修改/etc/squid/squid.conf文件。 把“http_access deny all”改成：
  1. #http_access deny all
  2. http_access allow all
复制代码


3. 启动squid应用

$ sudo systemctl enable squid
$ sudo systemctl start squid

4. 在火狐浏览器里设置代理 （Preferences -> Network Proxy -> Settings ...)。见图。 使用Linux机器的ip地址。 运行命令"ifconfig"查找ip地址。
