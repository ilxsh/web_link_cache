Python中图像的数组化存储方式

2017年09月20日 12:15:43 GZKPeng 阅读数：1190

一张720*250的RGB图片，个人感觉图片是说的长乘以宽（长720个像素宽250个像素），跟数组的m*n，m行n列还不一样，所以会以250*720*3的方式存在一个三维数组中(250行720列)，在python的Variable Explorer中，点开数组会显示720*3的二维数组，然后下方有index是显示行数这里是0~249。[:,0]代表R，[:,1]代表G，[:,2]代表B
但是注意PIL中的Image的load()函数，直接上代码

from PIL import Image
im = Image.open("xxx")
pix = im.load()
pix.[3,4] #这里却是指图像数组中的4行3列

这是因为Image.open()产生的是720*250的数据结构
