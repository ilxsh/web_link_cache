IP命令总结

如果设定固定的IP地址，那么在/etc/sysconfig/network-scripts/ifcfg-XXX以及/etc/resolv.conf中即可。

这里是学习ip命令的总结：

# ip --help
Usage: ip [ OPTIONS ] OBJECT { COMMAND | help }
　　　　ip [ -force ] -batch filename
where OBJECT := { link | address | addrlabel | route | rule | neigh | ntable | tunnel | tuntap | maddress | mroute | mrule | monitor | xfrm | netns | l2tp | macsec | tcp_metrics | token }
　　 OPTIONS := { -V[ersion] | -s[tatistics] | -d[etails] | -r[esolve] | -h[uman-readable] | -iec | -f[amily] { inet | inet6 | ipx | dnet | bridge | link } | -4 | -6 | -I | -D | -B | -0 | -l[oops] { maximum-addr-flush-attempts } | -o[neline] | -t[imestamp] | -ts[hort] | -b[atch] [filename] | -rc[vbuf] [size] | -n[etns] name | -a[ll] }

用到的比较多的主要就是ip link | ip addr | ip route 其他的用到再更新

1、ip link ：主要用于修改网络设备的配置信息，包括修改MAC地址，名称


# ip link help

　　　　　　　　 [ txqueuelen PACKETS ]
　　　　　　　　 [ address LLADDR ]
　　　　　　　　 [ broadcast LLADDR ]

# ip link show ： 查看端口的具体信息

# ip link add eth1 name Test type bridge ：创建一个名字是Test的类型是bridge的eth1端口
