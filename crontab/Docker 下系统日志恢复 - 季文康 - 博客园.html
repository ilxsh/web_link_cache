Docker 下系统日志恢复

众所周知，docker 是一款进程级虚拟机。上文我们已经分析解释了该如何使用，本文针对日志消失问题进行恢复与解释。

Docker 可以理解为阉割版的系统，内部功能不全。可以通过组件安装进行基本功能恢复。已恢复日志为例：安装 crontab 即可
