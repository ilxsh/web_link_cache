Linux 下 ftp的使用

最近需要在Linux上搭建FTP服务，通过网上的一些大神学习了一些新知识，在这个做一个总结：

Linux 下FTP 为 vsftp （linux red hat）
1、FTP配置路径：/etc/vsftpd/
ftp的配置文件主要有三个：
ftpusers 黑名单
user_list 可以是黑名单也可以是白名单
vsftpd.conf vsftpd的主配置文件
修改 vsftpd.conf 中 anonymous_enable=NO 为不允许匿名登录，限制默认的为FTP（密码也是FTP）的账号登录

2、FTP 服务路径：/usr/lib/systemd/system/ ，要进行FTP的启动、停止、重启需要先进入该目录:
检查FTP运行状态：service vsftpd status
启动FTP服务：service vsftpd start

停止FTP服务：service vsftpd stop

重启FTP服务：service vsftpd restart


3、创建FTP用户：
useradd ftpuser(用户名) -s /sbin/nologin --限定用户test不能telnet，只能ftp
passwd ftpuser(用户名) ，根据提示输入两次新的密码即可


4、设置FTP是否开启运行
FTP设置开机自动运行：chkconfig vsftpd on

关闭FTP开机自动运行：chkconfig vsftpd off

查看所有服务开启自动运行的情况：chkconfig --list
