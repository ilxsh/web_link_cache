超级详细傻瓜式图文通俗易懂教程使用EwoMail开源邮件系统在centos/Linux搭建邮件服务器

2019年11月21日 7:48 下午 作者：老梁 分类：技术专题  310

最近各种网站的迁移，各种的调试，我以前的服务器里是有邮局文件的，以前使用的是Winwebmail，具体请看站内 WinWebMail邮件服务器（WinWebMail Server 4.2.0.1 企业版本） 以前使用的是Windows系统，可以支持ASP的文件，现在使用了Linux系统，不能很好的支持ASP文件了，不得不换了一个邮件系统，找了一下，看到了新的邮件系统，那就是EwoMail开源邮件系统，看上去确实不错的说，既然不错，那就开始安装调试吧；

EwoMail官网

深圳智沃科技有限公司 http://ewomail.com/

安装准备

1、一台新装crntos7/8系统的服务器，最低配置1个cpu处理核心，1GB内存，40GB硬盘；（建议2G内存，免得麻烦，我直接腾讯云买的1H2G1M的 88RMB一年）

2、关闭selinux，vi /etc/sysconfig/selinux；（腾讯云，阿里云都可以忽略这一步）

关闭selinux

3、创建swap交换空间，官方介绍病毒检测程序没有swap空间无法启动（2G内存或者含2G，腾讯云，阿里云都可以忽略这一步）

站内教程 Linux系统内存不足时创建swap文件和分区

#创建1G的swap，可以根据你的服务器配置来调整大小
dd if=/dev/zero of=/mnt/swap bs=1M count=1024
#设置交换分区文件
mkswap /mnt/swap
#启动swap
swapon /mnt/swap
#需要修改文件 /etc/fstab 中的 swap 行，添加
/mnt/swap swap swap defaults 0 0

4、设置主机名，ewomail会自动使用邮箱域名前面加mail作为主机名，邮件主域名为laoliang.net，则设置服务器主机名为mail.laoliang.net

hostnamectl set-hostname mail.laoliang.net

主域名解析

文档使用的域名只是例子，只用于参考，实际过程中请使用你的域名哦；这里使用阿里云DNS为参考哦；

有一个@ MX的域名，请解析为mail.laoliang.net 改成你的域名即可，我没用mail，用的其他的，特此告知一下哦；

开放端口

端口都是TCP类型
8000，8010，8020，25，143，993，995，587，110，109，22，80，465

如使用的是云服务器，可能需要在控制面板的防火墙开放以上或者以下端口

必须打开的端口8000，8010，25，143

服务安装（操作以centos7作为演示）

安装前请服务器必须已链接网络，安装时间将会根据你的系统配置和网络环境大概会在10分钟内安装完成。（需要root权限）

打开：http://www.ewomail.com/list-11.html 输入你的域名获取安装代码

wget -c http://download.ewomail.com:8282/ewomail-1.05.sh && sh ewomail-1.05.sh laoliang.net

安装过程中可能会显示 shutting down postfix : FAILED，如果它的下面再出现一条 starting postfix : OK ，那就是正常的，可以不予理会；

安装成功后将会输出”Complete installation”。

查看安装的域名和数据库密码

cat /ewomail/config.ini

安装之后需要配置域名解析（我这是阿里云）

访问地址（将IP更换成你服务器IP即可）

邮箱管理后台：http://IP:8010 （默认账号admin，密码ewomail123）

进来需要设置的是邮箱系统设置，基本设置要都写成自己的哦；

web邮件系统：http://IP:8000 登录前台

测试腾讯邮局以及126邮局收取信件；

邮局测试收取邮件

此刻看的出收发邮件都是正常的，值得说明的是，腾讯云解封25端口比阿里云要方便，但不可以乱发垃圾邮件哦；使用邮件建议腾讯云；

集成以下软件:

Postfix：邮件服务器
Dovecot：IMAP/POP3/邮件存储
Amavisd：反垃圾和反病毒
Fail2ban：监控策略
LNAMP：apache2.2，nginx1.8， mysql5.5，php5.4
EwoMail-Admin：WEB邮箱管理后台
Rainloop：webmail
