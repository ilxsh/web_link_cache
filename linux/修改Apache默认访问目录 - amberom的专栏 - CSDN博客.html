修改Apache的默认站点目录

        Apache HTTP Server安装好后，默认的站点目录位于其安装目录下的htdocs文件夹内，默认首页是该文件夹的index.html文件。比如，我的Apache是安装在C:/Program Files/Apache Software Foundation/Apache2.2/内，那么我的Apache的完整的默认站点目录地址是C:/Program Files/Apache Software Foundation/Apache2.2/htdocs/，默认首页就是C:/Program Files/Apache Software Foundation/Apache2.2/htdocs/index.html。

    修改方法：

1、到Apache的安装目录下找到conf文件夹，该文件夹内会httpd.conf这样一个文本文档（见图1），它是Apache的配置文件，负责指挥Apache的运行。

                                (图1)

2、双击打开httpd.conf这个文本文档，然后同时按下键盘上的Ctrl+F组合键找“DocumentRoot "”（带英文双引号"，见图2），查找出来之后（仅有一处，在第177行处），将字符串“DocumentRoot”后面双引号内的字符串修改为自己想要设定的网站目录。

如修改为：DocumentRoot "C:/Program Files/Apache Software Foundation/Apache2.2/test"


                                    （图2）

3、完成上述第2步之后，暂不要关闭httpd.conf文件，继续查找，查找“<Directory "”（带英文双引号"，见图3），查找到“<Directory "”之后（在第204行处），将第一个“Directory”字符串后面双引号内的字符串修改为自己想要设定的网站目录。

如修改为：<Directory "C:/Program Files/Apache Software Foundation/Apache2.2/test">



                                （图3）

5、同时按下键盘上的Ctrl+S组合键保存上述修改，现在，在test文件夹新建一个html文档就会成为默认站点路径了，不过一定记得要停止（STOP）一次Apache服务然后再启动（START）（右键我的电脑，点击管理），使刚刚的修改生效。（注意：这是很多改了过后发觉没生效的重要原因）

修改的整体情况如下图


注意：test文件夹下html文档名必须为：index.html。如果不是这个名字，则浏览器打开时会默认打开本文件夹下的所有元素（包括图片和html文档）！
