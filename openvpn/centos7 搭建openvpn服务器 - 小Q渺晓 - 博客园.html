centos7 搭建openvpn服务器

1，简介

  • OpenVPN是一个开源代码的VPN应用程序，可让您在公共互联网上安全地创建和加入专用网络。相比pptp，openvpn更稳定、安全。
  • 本篇博客主要介绍下面两点：
    • 1 Centos 7下安装与配置OpenVPN；
    • 2 客户端连接OpenVPN服务器(window 、Ubuntu、 Ios、 Android)

2，安装依赖

  • 1 公网服务器IP或者国外VPS及root权限；
  • 2 由于OpenVPN在默认的CentOS软件库中不可用，需要安装EPEL；
yum install -y epel-release

4，OpenVPN Server 安装

4.1 安装OpenVPN

yum install openvpn easy-rsa -y

4.3 配置OpenVPN

  • OpenVPN在其文档目录中有示例配置文件，拷贝该示例配置文件到配置目录
cp /usr/share/doc/openvpn-*/sample/sample-config-files/server.conf /etc/openvpn
  • 修改该示例配置文件
vi /etc/openvpn/server.conf
  • 修改DNS服务器为google公共DNS服务器
;push "dhcp-option DNS 8.8.8.8"
;push "dhcp-option DNS 1.1.1.1"
  • 修改完后的配置文件

5，安全配置

  • 设置firewall规则
systemctl start firewalld.service

firewall-cmd --statefirewall-cmd --zone=public --list-all

firewall-cmd --add-service=openvpn --permanent

firewall-cmd --add-port=1194/udp --permanent

firewall-cmd --add-port=22/tcp --permanent

firewall-cmd --add-source=10.8.1.0 --permanent

firewall-cmd --query-source=10.8.1.0 --permanent

firewall-cmd --add-masquerade --permanent

firewall-cmd --query-masquerade --permanent

firewall-cmd --reload
  • 设置内核转发
# vim /etc/sysctl.conf
net.ipv4.ip_forward = 1
# sysctl -p
  • 禁止 ssh 登陆
cat >> /etc/hosts.allow << EOF
# 禁止 192.168.0.0/16 以外的ip登录
sshd: 192.168.*.*
sshd: ALL:deny
EOF
cat >> /etc/hosts.deny << EOF
# 禁止 192.168.0.0/16 以外的ip登录
sshd: ALL EXCEPT 192.168.*.*
EOF

7，openVPN 客户端

7.1 linux 客户端

7.1.1 目录结构

# tree /etc/openvpn/
/etc/openvpn/
├── client
│   ├── config
│   │   └── client.ovpn
│   └── logs
│       ├── client.log
│       └── client-status.log
├── easy-rsa
│   └── pki
│       ├── ca.crt
│       ├── openvpnclient.crt
│       ├── openvpnclient.key
│       └── ta.key
└── server

7.1.2 客户端配置文件

# cat > /etc/openvpn/client/config/client.ovpn << EOF
client
dev tun
proto udp
remote X.X.X.X 1194  ; X.X.X.X为服务器IP地址，或者域名 此处端口号必须与服务器中的端口号保持一致
resolv-retry infinite
nobind
persist-key
persist-tun
comp-lzo
verb 3
ca /etc/openvpn/easy-rsa/pki/ca.crt
cert /etc/openvpn/easy-rsa/pki/openvpnclient.crt   ;ubuntu上时必须使用绝对路径例如 /etc/openvpn/client/config/client.crt
key /etc/openvpn/easy-rsa/pki/openvpnclient.key   ;与上面一样，ubuntu上需要绝对路径
tls-auth /etc/openvpn/easy-rsa/pki/ta.key 1    ; 服务器需要设置为0， 客户端需要配置为1.
cipher AES-256-CBC    ; ubuntu和os x上面必须指定，否则会出现错误。
keepalive 10 120
route-delay 2
status /etc/openvpn/client/logs/client-status.log
log-append /etc/openvpn/client/logs/client.log
EOF

7.1.3 启动 openVPN 客户端

# openvpn --config  /etc/openvpn/client/config/client.ovpn
Enter Private Key Password: openclient.com

7.2 windows 客户端

7.2.1 下载客户端

https://github.com/OpenVPN/openvpn/archive/v2.4.6.zip

7.2.2 配置客户端

    1. 需要向管理员申请openvpn的配置及秘钥文件(总共包括5个文件：ca.crt证书、config.ovpn客户端配置文件、用户.crt、用户.csr、用户.key)。(其中，config.ovpn客户端文件是直接配置好拷贝给客户机的，这个文件在服务器端是不存在的，需要自己编写)
    1. 软件安装包见文末
    1. 如果你的电脑没有安装过TAP，安装过程中会有下面的提示，选择安装。
