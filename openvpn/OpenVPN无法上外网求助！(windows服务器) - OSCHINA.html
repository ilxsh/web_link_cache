OpenVPN无法上外网求助！(windows服务器)

livecpu 发布于 2017/07/24 11:46
阅读 2K+
收藏 0
答案 1
OpenVPN
开发四年只会写业务代码，分布式高并发都不会还做程序员？->>>   

先给大家说一下情况：

我是在windows ser2008 下搭建的openvpn的服务端，搭建过程不说了，就是官方指导的方法，搭建完毕之后，客户端也能正常连接。

于是我就尝试让客户端能上外网，在网上找了一篇帖子(http://baimei.blog.51cto.com/8340353/1423644),用他的方法添加了“路由和远程访问”

但是客户端还能不能正常上网。很是郁闷，已经自己鼓捣了一个星期了。今天特意来请教大家。

附上我的Server.ovpn配置:

port 1194
proto udp
dev tun
ca ca.crt
cert server.crt
key server.key
dh dh1024.pem

topology subnet
server 10.8.0.0 255.255.255.0

ifconfig-pool-persist ipp.txt

push "route 192.168.0.0 255.255.255.0"   #这是服务器内网网段
push "route 0.0.0.0 0.0.0.0"

push "redirect-gateway def1 bypass-dhcp"
push "dhcp-option DNS 223.5.5.5"
push "dhcp-option DNS 114.114.114.114"

client-to-client
duplicate-cn
keepalive 10 120

cipher AES-256-CBC

comp-lzo
persist-key
persist-tun

verb 3

和 client.ovpn配置


client
dev tun
proto udp

remote X.X.X.X 1194

resolv-retry infinite

nobind

persist-key
persist-tun

ca ca.crt
cert client.crt
key client.key

remote-cert-tls server

cipher AES-256-CBC

comp-lzo

verb 3

谢谢大家！
