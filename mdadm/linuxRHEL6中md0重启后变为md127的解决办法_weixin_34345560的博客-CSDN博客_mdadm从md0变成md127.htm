什么是单点登录？业务模块拆分之后登录会面临哪些问题？跨域单点登录如何用代码实现？这节课讲解跨域单点登录原理分析及项目实战

假如我们使用/dev/sdb1和/dev/sdc1两个分区建立了一个RAID1设备/dev/md0：


# mdadm  --create  /dev/md0  --level=1  --raid-devices=2  /dev/sd[bc]1

# mkfs.ext4 /dev/md0

重启后，/dev/md0设备就会自动变为/dev/md127设备。


解决办法：修改配置文件/etc/mdadm.conf第二列的设备名为md0。


# mdadm --detail --scan >> /etc/mdadm.conf

# vim /etc/mdadm.conf

引用

ARRAY /dev/md/godben:0 metadata=1.2 name=godben:0 UUID=c250f478:6e6de973:05a10768:503480be


将第二列内容即红色部分改为/dev/md/md0，如下所示：

ARRAY /dev/md/md0 metadata=1.2 name=godben:0 UUID=c250f478:6e6de973:05a10768:503480be


重启电脑后生效。
