【Git】git使用 - 冲突conflict的解决演示

冲突的解决

  （如果git使用不熟练）建议在push不了时，pull之前。在本地创建一个新的分支并commit到local，以保证本地有commit记录，万一出什么问题，可以找回代码，以免代码丢失。

  （更甚者，把整个文件夹备份，不然出现找不回代码那就开心了）
