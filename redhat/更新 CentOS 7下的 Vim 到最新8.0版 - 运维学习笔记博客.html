更新 CentOS 7下的 Vim 到最新8.0版

  • 博主： 明月登楼
  •  发布时间：2018 年 04 月 17 日
  •  5617 次浏览
  •   4 条评论
  •  624 字数
  • 分类： Linux
  1.  首页
  2. 正文  
    3. 分享到：

这几天明月因为需要更新 Hexo 博客，所以使用 Vim 的频率还是很高的，期间就发现 CentOS 7 上的 Vim 竟然是7.0的，而看到很多 Vim 教程、文章上的截图人家都是8.0的，使用yum update更新多次都没有发现有新的更新包，就怀疑是yum的更新源有问题，见天找到了一个 Vim 8 的更新源马上就添加了，一更新果然顺利升级 Vim 到了8.0.1699版，特发文笔记之。

添加 Vim 的更新源：

curl -L https://copr.fedorainfracloud.org/coprs/mcepl/vim8/repo/epel-7/mcepl-vim8-epel-7.repo -o /etc/yum.repos.d/mcepl-vim8-epel-7.repo

使用 yum 更新 Vim 到最新版本：

yum update -y vim*

仔细了解了一下才知道，其实 Vim 在2016年就已经更新到8.0了，不知道为啥 CentOS 里的竟然还是7.0的，据说7.0对不少 Vim 插件支持都不好，所以，这次果断更新最新版 Vim 。
