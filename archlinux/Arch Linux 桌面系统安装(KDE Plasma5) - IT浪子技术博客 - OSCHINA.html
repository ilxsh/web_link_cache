Arch Linux 桌面系统安装(KDE Plasma5)
原

 浪子Langzi 发布于 2018/09/05 10:42
字数 730
阅读 3.6W
收藏 3
点赞 1
评论 2
KDENetworkManagerPacmanPlasmaArch Linux

前提

已经安装了基本的系统，基本系统未安装的，传送门 : Arch Linux 基本系统安装教程( UEFI+GPT )

这里均采用最小安装

配置国内源

nano /etc/pacman.conf

文档末尾添加

[archlinuxcn]
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch

更新软件仓库

sudo pacman -Sy

后请安装 archlinuxcn-keyring 包以导入 GPG key。

pacman -S archlinuxcn-keyring

 

硬件接口服务

参考 xorg-server

pacman -S xorg-server

安装驱动

pacman -S xf86-video-intel  xf86-video-ati

xf86-video-intel intel显卡驱动

 xf86-video-ati  AMD显卡驱动

输入设备

pacman -S xf86-input-libinput // (xorg-server已经默认安装)

触摸板驱动

pacman -S xf86-input-synaptics

安装显示管理器

参考 SDDM  默认的主题很丑，可以更换主题

pacman -S sddm sddm-kcm

设置SDDM自启动否则无法启动桌面

systemctl enable sddm

如果不安装显示启动管理器（SDDM）就需要安装 xorg-xinit,否则无法启动桌面 参考 xorg-init

 

安装kde桌面

pacman -S plasma-desktop 

参考 Plasma-desktop

安装基本的软件

pacman -S kdebase

包括  dolphin    kate  kdialog   keditbookmarks    kfind   khelpcenter  konqueror konsole  kwrite

安装中文字体

pacman -S ttf-dejavu ttf-liberation wqy-microhei

wqy-microhei（温泉驿体）

安装声音软件包

pacman -S alsa-utils pulseaudio pulseaudio-alsa

安装中文输入法

pacman -S fcitx fcitx-rime fcitx-im kcm-fcitx fcitx-sogoupinyin

配置输入法

nano /home/<username>/.xprofile

export LANG=zh_CN.UTF-8
export LC_ALL=zh_CN.UTF-8
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"

安装网络工具

pacman -S  net-tools networkmanager plasma-nm

net-tools 提供了ifconfig命令 若不安装此软件可以使用ip address代替ifconfig

networkmanager 能够实现网络的自动链接

plasma-nm 系统托盘网络管理工具

自启动networkmanager 注意大小写

systemctl enable NetworkManager

SDDM主题

yaourt sddm-theme-kde-plasma-chili

需要安装开发环境的软件包括 autoconfig automake make fakeroot gcc pkgconf patch等

或者直接安装完整开发包base-devel即可 pacman -S base-devel

安装SSH

pacman -S openssh

启动ssh

systemctl start sshd

 

设置pacman彩色输出

打开/etc/pacman.conf文件，找到被注释的#Color，改为Color。pacman就会输出彩色信息，方便查看

图片浏览工具

pacman -S gwenview

识别window分区

pacman -S  ntfs-3g

主题

pacman -S arc-kde

浏览器

pacman -S google-chrome

首次用户目录生成

安装 xdg-user-dirs

pacman -S xdg-user-dirs

执行命令

xdg-user-dirs-update

若不慎删除了某个默认用户文件夹，使用如下命令强制重新生成家目录文件夹

xdg-user-dirs-update --force

该命令只对当前用户起作用

图标

pacman -S papirus-icon-theme

 

其他软件

ksystemlog //查看日志

ark    //压缩文件管理

kget //文件下载

kcalc //计算器

kcolorchooser //取色配色工具

print-manager //打印机管理

spectacle    // 截屏工具

GRUB2主题

yaourt -S  grub2-theme-vimix-git

安装完成后

sudo vim /etc/default/grub.cfg

GRUB_THEME="/boot/grub/themes/Vimix/theme.txt"

图形化包管理工具

pacman -S pamac-aur
